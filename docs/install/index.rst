Which Should I Pick?
--------------------

**TL;DR:** The :ref:`install_user` instructions are best for most
people.

Consult the table below to be sure:

.. list-table::
    :header-rows: 1

    * - Goal
      - Best Approach

    * - Make sprite sheets from fonts
      - :ref:`install_user`

    * - ``import fontknife`` in Python scripts
      - :ref:`install_user`

    * - Write or test bug fixes and new features
      - :ref:`install_contributor`
